Web Data Extractor

O Web Data Extractor é uma ferramenta projetada para a extração de dados textuais a partir de redes sociais, focando inicialmente na rede social Reddit.

Nosso objetivo é permitir a extração destes dados textuais a partir de palavras-chaves fornecidas pelo usuário na aba scraper. O mesmo terá a opção de realizar o download do seu dataset em JSON ou CSV, como também terá a opção de pesquisar entre os datasets já baixados em outros momentos, ou seja, eles serão persistidos.

Nossa missão é auxiliar pesquisadores e desenvolvedores com uma ferramenta que fomenta a Ciência Aberta. Acesse sua conta para aproveitar todas as oportunidades que temos para você!
