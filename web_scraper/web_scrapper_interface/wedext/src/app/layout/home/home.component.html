O Web Data Extractor (WeDEx) é uma ferramenta desenvolvida para facilitar a extração de dados a partir de redes sociais, com foco inicial na plataforma Reddit. Em versões futuras, outras redes também serão integradas.

Nosso principal objetivo é oferecer uma interface acessível e intuitiva auxiliando pesquisadores e pesquisadoras da área.

Atuamos com o propósito de apoiar a produção científica e tecnológica por meio da Ciência Aberta, fornecendo recursos que simplificam a coleta de dados públicos.
