O Web Data Extractor (Wedex) é uma ferramenta desenvolvida para facilitar a extração de dados textuais a partir de redes sociais, com foco inicial na plataforma Reddit. Em versões futuras, outras redes também serão integradas.

Nosso principal objetivo é oferecer uma interface acessível e intuitiva, permitindo que pesquisadores e pesquisadoras realizem estudos com dados legítimos, respeitando as regras e políticas de uso de cada plataforma.

Atuamos com o propósito de apoiar a produção científica e tecnológica por meio da Ciência Aberta, fornecendo recursos que simplificam a coleta de dados públicos.
