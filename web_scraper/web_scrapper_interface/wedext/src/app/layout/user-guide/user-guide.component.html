Guia do Usuário

A navegação na ferramenta Wedex é feita por meio das abas "Extrator" e "Dataset". Na aba Extrator, o usuário pode:

  • Inserir suas credenciais da API do Reddit
  • Definir palavras-chave e subreddits de interesse
  • Iniciar o processo de coleta de dados

Na aba Dataset, é possível:

  • Visualizar os dados extraídos (postagens e comentários)
  • Exportar os resultados em JSON ou CSV

Para utilizar o Wedex com a API do Reddit:

  1. Acesse a página de criação de aplicativos
  2. Crie um novo app do tipo script
  3. Use http://localhost:8080 como redirect URI
  4. Copie o client ID e o client secret exibidos
  5. Consulte a documentação da API do Reddit para mais detalhes
