Guia do Usuário

A navegação na ferramenta WeDEx é feita por meio das abas "Extrator" e "Dataset". Uma vez que esta aplicação utiliza a API do Reddit, para conseguir realizar a extração é necessário:

Criar suas credenciais na API do Reddit:

  1. Crie uma conta na referida rede social, caso não tenha.
  2. Acesse a página de criação de aplicativos para desenvolvedores clicando aqui.
  3. Crie um novo app do tipo script
  4. Nomeie seu app da forma que desejar e faça uma breve descrição do seu objetivo com este app
  5. O campo about url não é obrigatório
  6. Use http://localhost:8080 como redirect URI
  7. Clique em create app
  8. Copie o client ID e o client secret exibidos
  9. Consulte a documentação da API do Reddit para mais detalhes

Para realizar a extração, na aba Extrator:

  1. Insira o usuário e senha da sua conta no Reddit.
  2. Utilize o Client ID e o Client Secret, como mostrado anteriormente.
  3. O User Agent deve conter o nome da aplicação, sua versão e o nome de usuário do Reddit (ou um contato). Ex: WeDEx v1.0 by /u/seu_usuario
  4. Digite as palavras-chave que deseja buscar, separando por vírgula ou utilizando "OR"
  5. Adicione os subreddits que serão vasculhados, separados por vírgula
  6. Clique em "Extrair". O tempo de extração dependerá da quantidade de palavras-chave, subreddits e da velocidade da sua internet, que pode levar alguns minutos
  7. Ao finalizar a extração, você será redirecionado para a aba "Dataset", onde, ao clicar nos botões "Postagens" ou "Comentários", poderá visualizar os dados extraídos em formato de tabela e exportá-los em CSV ou JSON
