Sobre o Wedex

O Wedex foi criado com o compromisso de contribuir com a comunidade científica, oferecendo uma solução ética, segura e acessível para coleta de dados públicos nas redes sociais.

A ferramenta está em constante evolução e é construída com base nos princípios da Ciência Aberta, respeitando as legislações vigentes, como a Lei Geral de Proteção de Dados (LGPD).

Nossa ferramenta utiliza exclusivamente APIs públicas e oficiais, como a do Reddit, garantindo que todo acesso aos dados seja autorizado, documentado e feito dentro dos termos de uso. Para mais informações, consulte os termos da API de dados do Reddit.

Esta aplição faz parte de uma iniciativa do SIDE (Semantics, Intelligence and Data Ecosystems), grupo de pesquisa dedicado ao desenvolvimento de ecossistemas de dados com abordagens semânticas e inteligentes.
