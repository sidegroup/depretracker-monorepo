Sobre o Wedex

O WeDEx foi criado com o compromisso de contribuir com a comunidade científica oferecendo uma solução acessível para coleta de dados públicos nas redes sociais.

A ferramenta está em constante evolução e é construída com base nos princípios da Ciência Aberta, respeitando as legislações vigentes, como a Lei Geral de Proteção de Dados (LGPD).

Nossa ferramenta utiliza exclusivamente APIs públicas e oficiais, como a PRAW Reddit. Para mais informações, consulte os termos da API de dados do Reddit.

Esta aplição faz parte de uma iniciativa do SIDE (Semantics, Intelligence and Data Ecosystems), grupo de pesquisa dedicado ao desenvolvimento de aplicações baseadas em dados e em Inteligência Artificial
